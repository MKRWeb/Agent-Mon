# Monad Agent Hub Chatbot

A native, hyper-optimized AI chatbot interface designed for the Monad ecosystem. 

## Features
- **Monad UI/UX**: Deep dark mode with Monad Purple accents and glitch animations.
- **Local History**: Chat history is saved in the browser's `localStorage`.
- **Hybrid Brain**: Responds instantly to hardcoded Monad-specific keywords (founders, lore, tech specs), and falls back to an AI API for broader questions.
- **Modularized Code**: Separated HTML, CSS, and JS for easy feature expansion.
