# Monad Agent Hub Chatbot 💜

A native, hyper-optimized AI chatbot interface designed for the Monad ecosystem. 

**🟢 Live Demo:** [https://agent-mon.vercel.app/](https://agent-mon.vercel.app/)

## Features
- **Monad UI/UX**: Deep dark mode with Monad Purple accents, custom scrollbars, and glitch animations.
- **Local History**: Chat history is saved in the browser's `localStorage` for persistent sessions.
- **Hybrid Brain**: Responds instantly to hardcoded Monad-specific keywords (founders, lore, tech specs, 10k TPS), and falls back to an AI API for broader or conversational queries.
- **Modularized Code**: Clean separation of HTML, CSS, and JS for easy feature expansion and maintenance.
